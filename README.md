# Seesaw Simulation

A web-based simulation that demonstrates the physical concept of torque and balance using a seesaw mechanism. The project features real-time physics calculations, state persistence.

### 1. Design & User Experience
- **Modern & Minimalist Layout:** My primary goal was to build a modern design structure. I preferred a minimalist layout centered on the page to ensure that the user can see the entire simulation, statistics, and history at a **single glance** without needing to scroll or search for information.
- **Visual Consistency (Color Coding):** I placed significant importance on using the weight colors within the UI design. The "Next Weight" indicator and the borders of the "History" items are dynamically styled to match the color of the generated weight. This creates a visual link between the data and the physical objects on the plank.
